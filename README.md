# Drift

A personal system of thinking, built from lived patterns.

Drift is a digital garden where ideas are short, dense, and interlinked. Each article is a pattern observed through life. Articles link to each other so that meaning accumulates as you read deeper.

Live at [thamizh5197.github.io/drift](https://thamizh5197.github.io/drift/)

---

## Structure

- `content/articles/` — all articles as markdown files
- `data/tags.yaml` — tag vocabulary
- `static/` — CSS and inline expansion JS
- `layouts/` — Hugo templates
- `.github/workflows/` — auto-deploy to GitHub Pages on push

## Stack

Built with [Hugo](https://gohugo.io). Deployed via GitHub Actions to GitHub Pages. No dependencies, no frameworks.

## Writing a new article

```
hugo new content/articles/your-article-title.md
```

Then write, push, and the site rebuilds automatically.
